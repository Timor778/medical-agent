医疗问诊 Agent

这是一个基于 LangGraph 的 Web 版医疗问诊项目。页面支持提问、查看分诊结果、查看静态图和本次执行路径，并使用固定尺寸的可滚动结果区域。

发起问诊

建议使用不同的 `thread_id` 测试不同会话。这个系统用于医疗信息辅助，不替代医生面诊。

项目能力

1. 医疗意图识别与风险分级
2. 联网搜索真实资料
3. 搜索失败时自动重写查询词
4. 搜索仍失败时回退保守建议
5. 可视化展示节点流向和边的选择
