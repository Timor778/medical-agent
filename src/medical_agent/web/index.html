医疗问诊 Agent

这是一个基于 LangGraph 的 Web 医疗问答示例。页面支持提问、查看风险分级、查看真实运行路径、 查看静态图结构，并展示最终回答的流式输出。

发起问诊

它相当于一条会话的编号。
相同的 thread_id 会复用同一条线程上下文。
默认值 web-demo 只是前端预填的演示名字，没有特殊权限。
系统会优先尝试本地 RAG，再根据需要联网检索。 这个系统用于医疗信息辅助，不替代医生面诊。

项目能力

1. 医疗意图识别与风险分级
2. 本地 RAG 命中与联网检索
3. 检索失败时自动重写查询词
4. 真实运行流程可视化
5. 最终回答 token 流式输出
